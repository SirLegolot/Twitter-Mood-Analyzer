Project for 112.me - Twitter Mood Analyzer
Submitted to Hack 112
Team Members - Neel Gandhi(ngandhi), Darwin Torres Romero(dtorresr), Saisiddarth Domala(sdomala)


Files:
Images folder - contains the images used in this project
112dotme UI.py - Run this code to open the application
emotionAnalyzer.py  - Contains the code that integrates the twitter and watson outputs.
graphics.py - contains the code that outputs the emotion graph
twitterFns - code to set up api and define function to call all tweets of a user
watsonFns - code to set up watson developer cloud and define the tone analyzer function.
